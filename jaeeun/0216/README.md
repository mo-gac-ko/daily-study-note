###

#### 컴포넌트의 라이프사이클

      Mount → Update → Unmount

###

1. Mount : 페이지에 컴포넌트가 나타남

#####

2.  Update : 컴포넌트 정보를 업데이트

            - props가 바뀔 때
            - state가 바뀔 때
            - 부모 컴포넌트가 리렌더링 될때
            - this.forceUpdate로 강제로 렌더링을 트리거할 때

#####

3. Unmount : 페이지에서 컴포넌트가 사라짐

###

❗️ **라이프사이클 메서드는 _<u>클래스형 컴포넌트</u>_ 에서만 사용할 수 있음**
(함수형 컴포넌트에서는 hooks 기능을 활용)
